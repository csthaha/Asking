# q1.
 > js 全局执行为我们创建了两个东西：全局对象 和 this关键字
 - new 的实现原理：
    1. 创建一个空对象，构造函数中的this指向这个空对象
    2. 这个新对象被执行[原型]连接
    3. 执行这个构造函数，属性和方法添加到this引用的对象中
    4. 判断构造函数是否返回一个对象
        没有： 那么就返回这个this，即，创建的新对象
        有：返回构造函数中返回的对象。
